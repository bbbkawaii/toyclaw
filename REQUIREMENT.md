1. 项目概述
ToyGlobal AI Agent 是一款专为中小型玩具制造企业设计的出海辅助系统。该系统利用多模态人工智能技术，解决传统玩具工厂在拓展海外市场时遇到的文化壁垒、设计改款成本高、市场洞察缺失以及跨语言沟通等业务痛点。

用户通过上传单张玩具图片并选择目标市场，系统即可自动化输出包含外观改款建议、文化合规审查、本地化营销文案及市场趋势的数据报告，从而大幅降低产品出海的试错成本与周期。

2. 系统架构概览
系统采用前后端分离架构，核心业务逻辑由多模态 AI Agent 驱动。整体系统划分为六个核心微服务模块。

图像输入模块：接收用户上传的图片，进行预处理并提取玩具的形状、颜色、材质和基础风格等物理特征。

跨文化分析模块：通过检索内部构建的全球文化与宗教禁忌知识库，匹配目标市场的节日热点，并排查潜在的文化冲突。

改款建议模块：接收图像特征与文化分析结果，输出具体的颜色、造型调整方案，并调用图像生成 API 渲染改款后的 2D 或 3D 预览图。

合规与认证模块：根据选择的出口国家/地区，映射对应的行业合规标准（如欧盟 CE、美国 ASTM），提供材料安全与年龄分级建议。

销售赋能模块：根据提取的产品卖点与目标市场语言，自动生成本地化的产品描述、Slogan 及外贸商务沟通邮件模板。

市场洞察模块：整合外部数据源，输出目标市场的近期热销品类分析及定价区间参考。
